Camera System & NAS Storage

Four Raspberry Pi Zero 2 W units, each fitted with a camera module and weatherproof enclosure, stream continuous footage to a NAS with a RAM upgrade for improved throughput. Recordings are backed up locally on the NAS with a secondary offsite backup for redundancy.

AI Person Detection

A Google Coral USB TPU accelerates on-device inference for real-time person detection via Frigate NVR, keeping all video processing local with zero cloud dependency. Alerts trigger only on confirmed person events, dramatically reducing false positives.

Network-wide Ad & Threat Blocking

Pi-hole runs as the local DNS server for the entire network, blocking ads, trackers, and known malicious domains at the DNS layer before any device makes a connection. Paired with curated blocklists and a custom allowlist, it covers every device on the LAN without per-device configuration.

Remote Access via DDNS & Reverse Proxy

A dynamic DNS record keeps the home IP reachable under a stable hostname. An Nginx reverse proxy with HTTPS termination (Let’s Encrypt) fronts the camera dashboard and home automation interface, exposing them securely over the public internet without opening raw ports to internal services.

Reverse proxy & HTTPS setup

Honeypot & Passive Packet Monitoring

A dedicated Raspberry Pi runs both a honeypot server to detect and log unauthorized access attempts and a passive packet monitoring daemon that taps all network equipment, surfacing anomalous traffic patterns and lateral movement attempts before they reach production devices. Pairs with Pi-hole DNS logs for a unified threat picture.
